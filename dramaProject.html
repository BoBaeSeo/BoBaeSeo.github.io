프로젝트 소개

상영중인 연극 리스트를 보여주고 고객이 원하는 연극을 예매할 수 있는 페이지 제작

주요역할 및 담당

공통역할 :
1. 각각 생각해온 아이디어로 진행된 회의를 통해 프로젝트 기획 및 총괄
2. 총괄 후 데이터베이스 정의, 설계 및 구현,
3. 프로젝트 시작하기 전 개개인의 역량에 맞춰 역할을 분배
4. 데이터베이스에 필요한 자료를 다 함께 수집하여 방대한 데이터를 수집함.
5. bootctrap을 사용하지 않고 다 함께 css를 처음부터 작성하여 그 틀에 맞춰 코딩을 시작함
6. 서로 작성한 코드들을 하나로 합치고 그에 대한 에러코드 및 오류를 검출해 수정함
7. 서로 작업한 현황을 공유하고 현황에 따라 스케줄이나 사양을 변경함
8. 모든 프로젝트 관련 데이터베이스와 서류를 보관하고 관리함

엔지니어:
1. 로그인, 회원가입, 회원 수정 기능 구현

2. 회원 등급 기능
- 회원 티켓 구매 내역에 따라 등급이 바뀔 수 있도록 작업.
- 처음 회원 가입을 했을 경우 D등급, 2만원 이상 C등급, 7만원 이상 B등급 12만원이상 A등급 20만원 이상 S등급으로 나누어 고객이 티켓을 구매하거나 취소하는 경우를 모두 고려해 고객에게 등급을 매기고 그 등급을 이미지로하여 구현.

3. 예매내역 리스트 페이지 제작
- 로그인 된 고객이 예매한 내역을 볼 수 있는 페이지를 구현.
- DAO에서 쿼리문을 2개 사용하여 먼저 고객 코드를 select후 select된 고객 코드를 이용하여 티켓 테이블에서 예매 내역을 출력.
- 예매 내역이 존재하지 않는 경우에는 예매 내역이 존재하지 않는다는 문구를 출력.
- 예매내역에서 연극 제목을 누르면 연극 상세보기 페이지로 이동.

4. 검색 기능
- header에 input 태그를 이용하여 검색 기능을 만듦.
- like 쿼리문을 이용하여 검색하는 글자와 일치하는 페이지를 보여줌.

5. 연극 상세보기 페이지 제작
- 연극에 대한 내용과 연극을 관람한 고객들의 리뷰를 볼 수 있는 페이지를 구현.
- 오른쪽 사이드에 존재하는 리뷰 칸에 리뷰 내용과 별점을 남길 수 있고, 별점을 남기는 즉시 연극의 총 별점이 갱신.
- 연극 날짜와 예매 개수를 선택하여 연극을 예매 할 수 있도록 구현.

개발도구

- DB 설계 : Oracle SQL Developer
- IDE 도구 : Eclipse, Visual Studio code
- 웹 서버 : Apache Tomcat 9.0
- 언어 : Java, JavaScript, Html, CSS, JSP
- 주요 라이브러리 : jQuery, JSON, AJAX
